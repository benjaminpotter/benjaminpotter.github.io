## Methodology
I set out on this project to provide Dungeons and Dragons players with a simple and effective tool to generate realistic maps for their worlds.

My design choices were largely governed by the following criteria:
1. Interface is easy to use
2. Generated maps should require no additional changes by the user (for use on-the-fly)
3. Extensive map customization

These criteria were important to provide the user with a pick-up and go experience. I wanted the application to provide an awesome starting place for any map.

## What does the map maker include?
As the project grows, more features will be added. If you have a request for a feature, you can create a GitHub issue or email me at baffqd@gmail.com.

### v1.0 - Initial release

**Generation**
- Generate terrain from preset 'biome' types
- Place different terrain types e.g. forest, plains, mountains
- Place points of interest
- Connect points of interest by roads

**Customization**
- Edit names of points of interest
- Edit height of terrain with brush
